# Multi-Platform Chat TTS (Thai)

โปรเจกต์สำหรับดึงคอมเมนต์จาก Live Chat แพลตฟอร์มต่างๆ และแปลงเป็นเสียงพูดภาษาไทยอัตโนมัติ (Text-to-Speech) โดยใช้ Edge TTS ออกแบบมาเพื่อสตรีมเมอร์และคอนเทนต์ครีเอเตอร์ที่ต้องการระบบอ่านแชทภาษาไทยที่ลื่นไหลและเป็นธรรมชาติ

## 🚀 ฟีเจอร์และความสามารถ (Features & Capabilities)

- **รองรับ 4 แพลตฟอร์มหลัก:** 
  - **YouTube:** รองรับทั้ง Video ID และ URL ปกติ
  - **Facebook:** รองรับ Facebook Live URL
  - **Twitch:** เชื่อมต่อผ่านชื่อ Channel (รองรับ Anonymous Read-only)
  - **TikTok:** รองรับการดึงแชทจาก TikTok Live โดยตรง (ใช้เพียงชื่อผู้ใช้ @username)
- **ระบบเสียงภาษาไทยคุณภาพสูง:**
  - ใช้ **Edge TTS** (Microsoft Azure) ให้เสียงที่เหมือนมนุษย์
  - มีเสียงให้เลือก (เช่น Premwadee, Niwat)
  - **ระบบ Fallback:** หาก Edge TTS ขัดข้อง ระบบจะสลับไปใช้ **gTTS (Google TTS)** ให้โดยอัตโนมัติเพื่อให้การทำงานไม่สะดุด
- **การจัดการเสียงที่ชาญฉลาด (Smart Audio Management):**
  - **Queue System:** ระบบจัดการลำดับการอ่าน ป้องกันเสียงซ้อนกัน
  - **Dynamic Delay:** คำนวณเวลาหยุดหลังอ่านจบตามจำนวนตัวอักษร (ปรับแต่งได้) เพื่อความต่อเนื่อง
  - **Auto-Cleanup:** ล้างไฟล์เสียงชั่วคราวทิ้งทันทีหลังเล่นจบ และล้างโฟลเดอร์ Cache ทุกครั้งที่เริ่มโปรแกรมใหม่
- **อินเทอร์เฟซการใช้งาน:**
  - **GUI (CustomTkinter):** หน้าจอสวยงาม ใช้งานง่าย ปรับแต่งค่าได้เรียลไทม์
  - **CLI Mode:** สำหรับสาย Console หรือรันบน Server
  - **Real-time Logs:** ดูสถานะการทำงานและข้อผิดพลาดได้ผ่านหน้าจอโปรแกรม
- **ความเสถียร:**
  - ระบบ Auto-Reconnect พยายามเชื่อมต่อใหม่หากการดึงแชทหลุด
  - แยก Thread การทำงานชัดเจน (Collector, Generator, Player) ทำให้หน้าจอไม่ค้าง

## 🛠 การติดตั้ง

1. ติดตั้ง Python (แนะนำเวอร์ชัน 3.10 ขึ้นไป)
2. ติดตั้ง Library ที่จำเป็น:
```bash
pip install -r requirements.txt
```

## 💻 วิธีใช้งาน

### แบบใช้หน้าจอ GUI (แนะนำ)
1. รันไฟล์ `run_gui.py`:
```bash
python run_gui.py
```
2. เลือกแพลตฟอร์มที่ต้องการ (ติ๊กถูกหน้า Enabled)
3. ใส่ข้อมูลของ Live นั้นๆ:
   - **YouTube:** URL หรือ Video ID
   - **Facebook:** URL ของ Live
   - **Twitch:** ชื่อ Channel
   - **TikTok:** ชื่อผู้ใช้ (เช่น @username)
4. เลือกเสียง และตั้งค่า Delay ตามต้องการ 
5. กด **Start System**

### แบบ Command Line (CLI)
1. ตั้งค่าใน `config.ini`
2. รันไฟล์ `run_cli.py`:
```bash
python run_cli.py
```

## 📁 โครงสร้างโปรเจกต์
- `run_gui.py`: โปรแกรมหลักแบบหน้าจอ (รวมทุกแพลตฟอร์ม)
- `run_cli.py`: โปรแกรมหลักแบบ Command Line
- `engine.py`: หัวใจหลักในการดึงแชทและประมวลผล (Core Engine)
- `logger.py`: ระบบบันทึก Log
- `temp_audio/`: โฟลเดอร์เก็บไฟล์เสียงชั่วคราว
- `msg_queue/`: โฟลเดอร์พักข้อความ
