# YouTube Chat TTS (Thai)

โปรเจกต์สำหรับดึงคอมเมนต์จาก YouTube Live Chat และแปลงเป็นเสียงพูดภาษาไทยอัตโนมัติ (Text-to-Speech) โดยใช้ Edge TTS

## 🚀 ฟีเจอร์
- ดึงแชทจาก YouTube Live อัตโนมัติ
- แปลงเป็นเสียงพูดภาษาไทยคุณภาพสูง (เลือกได้ทั้งเสียงผู้หญิงและผู้ชาย)
- มีหน้าจอ GUI สำหรับตั้งค่าและควบคุมได้ง่าย
- จัดลำดับการเล่นเสียง (Queue) ไม่ให้เสียงซ้อนกัน
- ล้างไฟล์ขยะอัตโนมัติเมื่อเริ่มโปรแกรม

## 🛠 การติดตั้ง

1. ติดตั้ง Python (แนะนำเวอร์ชัน 3.10 ขึ้นไป)
2. ติดตั้ง Library ที่จำเป็น:
```bash
pip install -r requirements.txt
```

## 💻 วิธีใช้งาน

### แบบใช้หน้าจอ (แนะนำ)
1. รันไฟล์ `run_gui.py`:
```bash
python run_gui.py
```
2. ใส่ URL หรือ Video ID ของ YouTube Live
3. เลือกเสียง และกด **Save Settings**
4. กด **Start System**

### แบบ Command Line
1. แก้ไขค่าใน `config.ini` (ถ้ามี)
2. รันไฟล์ `run_cli.py`:
```bash
python run_cli.py
```

## 📁 โครงสร้างโปรเจกต์
- `run_gui.py`: โปรแกรมหลักแบบหน้าจอ
- `run_cli.py`: โปรแกรมหลักแบบ Command Line (แบบรวม)
- `engine.py`: หัวใจหลัก (ใช้ร่วมกัน)
- `collector.py`: ส่วนดึงแชท (แยกไฟล์)
- `generator.py`: ส่วนแปลงเสียง (แยกไฟล์)
- `player.py`: ส่วนเล่นเสียง (แยกไฟล์)
- `config.ini`: ไฟล์เก็บค่าตั้งค่าต่างๆ

## ⚠️ หมายเหตุ
- ไฟล์เสียงจะถูกเก็บชั่วคราวในโฟลเดอร์ `temp_audio` และจะถูกลบออกเมื่อเล่นจบ
- โปรแกรมนี้ใช้ `edge-tts` ซึ่งจำเป็นต้องมีการเชื่อมต่ออินเทอร์เน็ตขณะใช้งาน
