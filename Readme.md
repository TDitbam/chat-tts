# Multi-Platform Chat TTS (Thai)

โปรเจกต์สำหรับดึงคอมเมนต์จาก Live Chat แพลตฟอร์มต่างๆ และแปลงเป็นเสียงพูดภาษาไทยอัตโนมัติ (Text-to-Speech) โดยใช้ Edge TTS

## 🚀 ฟีเจอร์
- **รองรับหลายแพลตฟอร์ม:** YouTube, Facebook, Twitch และ **TikTok**
- ดึงแชทสดอัตโนมัติ
- แปลงเป็นเสียงพูดภาษาไทยคุณภาพสูง (เลือกได้หลายเสียง)
- มีหน้าจอ GUI สำหรับตั้งค่าและควบคุมได้ง่าย
- ระบบจัดการลำดับการเล่นเสียง (Queue) ป้องกันเสียงซ้อน
- ระบบ Fallback (gTTS) หาก Edge TTS ขัดข้อง

## 🛠 การติดตั้ง

1. ติดตั้ง Python (แนะนำเวอร์ชัน 3.10 ขึ้นไป)
2. ติดตั้ง Library ที่จำเป็น:
```bash
pip install -r requirements.txt
```

## 💻 วิธีใช้งาน

### แบบใช้หน้าจอ (แนะนำ)
1. รันไฟล์ `run_gui.py`:
```bash
python run_gui.py
```
2. เลือกแพลตฟอร์มที่ต้องการใช้งาน (ติ๊กถูกหน้า Enabled)
3. ใส่ข้อมูลของ Live นั้นๆ:
   - **YouTube:** URL หรือ Video ID
   - **Facebook:** URL ของ Live
   - **Twitch:** ชื่อ Channel
   - **TikTok:** ชื่อผู้ใช้ (เช่น @username)
4. เลือกเสียง และตั้งค่า Delay ตามต้องการ
5. กด **Start System**

### แบบ Command Line
1. แก้ไขค่าใน `config.ini`
2. รันไฟล์ `run_cli.py` หรือ `collector.py` (สำหรับ YouTube เฉพาะจุด)

## 📁 โครงสร้างโปรเจกต์
- `run_gui.py`: โปรแกรมหลักแบบหน้าจอ (รวมทุกแพลตฟอร์ม)
- `engine.py`: หัวใจหลักในการดึงแชทและประมวลผล
- `collector.py`: ส่วนดึงแชทเฉพาะ YouTube (Standalone)
- `logger.py`: ระบบบันทึก Log
- `temp_audio/`: โฟลเดอร์เก็บไฟล์เสียงชั่วคราว
- `msg_queue/`: โฟลเดอร์พักข้อความ (สำหรับ Collector แยกส่วน)

## ⚠️ หมายเหตุ
- โปรแกรมนี้จำเป็นต้องมีการเชื่อมต่ออินเทอร์เน็ตขณะใช้งาน
- สำหรับ TikTok: แนะนำให้ใช้ชื่อผู้ใช้ที่มี @ นำหน้าเพื่อให้ระบบค้นหาได้แม่นยำขึ้น
