Chat TTS v2.2.0 - Auto-Translate & Multi-Platform Update 🚀

[บันทึกการอัปเดต / Release Notes]

เวอร์ชัน v2.2.0 นี้มีการอัปเดตใหญ่เพื่อเพิ่มความสะดวกให้กับสตรีมเมอร์ในการรองรับผู้ชมจากต่างประเทศและปรับปรุงความเสถียรของระบบ

ฟีเจอร์ใหม่:
1. ระบบแปลภาษา Real-time (Auto-Translate): 
   - เพิ่มตัวเลือกแปลคอมเมนต์ภาษาต่างประเทศเป็นภาษาไทยโดยอัตโนมัติก่อนอ่านออกเสียง
   - ใช้ระบบ Google Translate (ไม่ต้องใช้ API Key)
2. ระบบเสียงสำรอง (Fallback System): 
   - หาก Edge TTS มีปัญหา ระบบจะสลับไปใช้ gTTS (Google) ให้โดยอัตโนมัติเพื่อให้การอ่านไม่สะดุด
3. การจัดการคิวเสียง (Smart Queue): 
   - ปรับปรุงการคำนวณดีเลย์ตามจำนวนตัวอักษรและป้องกันเสียงอ่านซ้อนกัน
4. รองรับ 4 แพลตฟอร์มหลัก:
   - YouTube, Facebook, Twitch และ TikTok Live

หมายเหตุสำหรับการใช้งานแบบ EXE:
- ไฟล์ ChatTTS.exe จะเปิดหน้าจอ Console (CMD) ไว้เสมอเพื่อโชว์สถานะการทำงาน (Log) ตามที่คุณต้องการ
- วิธีใช้: แตกไฟล์ ZIP แล้วรัน ChatTTS.exe ได้ทันที
